## 项目初衷
一款社团管理系统。

## 环境要求
* macOS or Linux or Windows
* JDK1.8
* Mysql
* Npm
* Vue.js
## 项目技术栈
* 后端 : SpringBoot SpringMVC Mybatis
* 前端 : Vue.js Element-UI Bootstrap
